## 本轮已处理

| 项目 | 状态 | 说明 |
|---|---|---|
| 单文件维护成本高 | 已完成 | 拆为 `index.html`、`styles.css` 和 `src/*.js` 原生 ES modules；无构建、无运行时依赖 |
| 空日继承前一天标签 | 已修复 | `src/stats.js` 以本地自然日 00:00 为硬边界，空日不再累计昨日最后标签 |
| 首条记录前统计口径 | 已修复 | 某天有第一条记录时，00:00 到第一条之间计为未记录 |
| 周/月/年跨日延续 | 已修复 | 汇总按每日独立统计累加，不跨日继承上一条记录 |
| 跨日长段确认绑定 | 已修复 | 无同日下一条记录时，历史日使用本地日边界作为 `endTs` |
| 小屏加记录后横向滚动 | 已修复 | 记录行、摘要行、ruler 文案和 footer 增加收缩/换行策略 |
| 周视图小屏周期标题溢出 | 已修复 | 周标题使用短格式显示，完整日期保留在 `aria-label` |
| hidden tooltip 撑宽页面 | 已修复 | tooltip pseudo 元素默认不生成盒子，仅 hover/focus 时显示 |
| 响应式回归测试缺失 | 已补充 | 新增 Playwright UI smoke，覆盖 320/375/430/768 宽度和三种数据状态 |

## 暂不进入本轮

| 项目 | 原因 |
|---|---|
| 云同步 / 账号 / 后端 | 超出本地 PWA 边界，会改变隐私和复杂度模型 |
| 更多报表或图表 | 当前目标是记录和复盘，不扩展分析产品能力 |
| 标签体系重做 | 真实记录样本不足时容易过度设计 |
| 提醒 / 自动追踪 | 会引入新的权限、隐私和误报问题 |
| 构建系统 / 前端框架 | 当前原生模块足够，构建会增加部署和缓存复杂度 |

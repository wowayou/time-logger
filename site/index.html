时间尺

5 秒记下真实做了什么——本地、离线、可信的一天时间线

打开应用

免费 · 无账号 · 数据只在你的设备上

它是什么

一个单页离线 PWA：按时间段记录一天做了什么，按「主线 / 维持 / 偏航 / 未记录」四桶统计时间去向，支持补录、区间编辑、跨日记录和计划。零运行时依赖、无构建、无后端——打开网页即用。

数据边界

  • 所有记录只存在浏览器本地（localStorage），不上传、不同步、无账号。
  • 本站不加载任何统计、广告或第三方脚本。
  • 完整备份可随时复制、下载或分享为 JSON；换设备或换浏览器用「导出 → 导入」迁移，导入是按记录逐条校验的安全合并。
  • 清除浏览器站点数据会删除记录——请定期导出备份。

安装到主屏幕（可选）

  • iPhone / iPad：在 Safari 打开 应用 → 分享 → 「添加到主屏幕」，即可离线全屏使用。
  • 桌面 Chrome / Edge：地址栏右侧「安装」图标。
  • 不安装也可以直接在浏览器里使用，功能相同。

从旧地址迁移

如果你此前在 wowayou.github.io/time-logger/ 使用时间尺：浏览器数据按域名隔离，记录不会自动跟过来。在旧地址「···」菜单里「存储备份」导出完整 JSON，然后在新应用「···」菜单里「导入」即可，导入前会逐条校验并预览冲突。
